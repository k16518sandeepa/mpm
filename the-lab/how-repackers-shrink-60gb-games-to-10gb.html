Repackers ලාගේ රහස: 60GB game එකක් 10GB කරන්නේ කොහොමද?

ඔයාලා Computer Games (PC Games) play කරන කෙනෙක් නම්, අනිවාර්යයෙන්ම FitGirl හෝ DODI වගේ නම් අහලා ඇති. Internet එකෙන් games download කරද්දී ගොඩක් අය හොයන්නේ මේ අයගේ රිලීසස් (Releases). ඒකට ප්‍රධානම හේතුව තමයි original game එක 60GB හෝ 100GB වගේ ලොකු සයිස් එකක් වුනත්, මේ අය ඒක 10GB හෝ 20GB වගේ පුදුමාකාර විදිහට size අඩු කරලා දෙන එකයි.

මේක ඇත්තටම මැජික් එකක්ද? කොහොමද මෙයාලා මේ තරම් ලොකු file එකක් මේ තරම් පොඩි කරන්නේ? අද අපි කතා කරන්නේ ඒ පිටුපස තියෙන තාක්ෂණික රහස ගැනයි.

1. Repackers සහ Crackers ලා කියන්නේ දෙගොල්ලක්.

ගොඩක් අය හිතාගෙන ඉන්නේ FitGirl හෝ DODI කියන්නේ games ක්‍රැක් කරන (Hacker/Cracker) අය කියලා. හැබැයි ඒක වැරදි මතයක්. මේ අය කරන්නේ වෙනත් Scene Groups (Scene Groups ගැන දැන ගන්න Click කරන්න) (උදාහරණයක් විදිහට RUNE, Skidrow හෝ Empress) වලින් crack කරපු ගේම් එකක් අරගෙන, ඒක තමන්ගේම ක්‍රමවේදයකට අනුව Highly Compress කරන එක විතරයි.

2. 60GB ගේම් එකක් 10GB කරන්නේ කොහොමද?

මේගොල්ලෝ මේ වැඩේ කරන්න ප්‍රධාන ක්‍රම කිහිපයක් පාවිච්චි කරනවා:

  • භාෂා පැකේජ ඉවත් කිරීම (Language Packs Stripping):
    අලුතින් එන ලොකු ගේම්ස්වල ඉංග්‍රීසි language එකට අමතරව ප්‍රංශ, ජර්මන්, ජපන්, රුසියානු වගේ තවත් language ගණනාවක හඬකැවීම් (Voice files) අඩංගු වෙනවා. මේ එක language පැකේජ් එකක් GB ගණනාවක් ලොකුයි. Repackers ලා කරන්නේ මේ හැම language එකක්ම වෙන වෙනම වෙන් කරන එකයි. එතකොට අපිට අවශ්‍ය ඉංග්‍රීසි language එක විතරක් download කරගෙන ඉතිරි GB ගණනාවම ඉතුරු කරගන්න පුළුවන්.
  • Video Files Compress කිරීම (Video Re-encoding):
    ගේම්ස් ඇතුළේ තියෙන Cutscenes videos ගොඩක් වෙලාවට තියෙන්නේ 4K හරි High Bitrate කොලිටියෙන්. මේවායේ සයිස් එක ගොඩක් වැඩියි. සමහර Repackers ලා මේ වීඩියෝස්වල කොලිටිය පොඩ්ඩක් අඩු කරලා (Encode කරලා) ඒවායේ සයිස් එක 50%කට වඩා අඩු කරනවා.
  • Advanced Compression Algorithms:
    අපි සාමාන්‍යයෙන් file එකක් Zip කරන්න පාවිච්චි කරන WinRAR හෝ 7-Zip වලට වඩා advanced algorithms මෙයාලා පාවිච්චි කරනවා. උදාහරණයක් විදිහට LZMA, Zstd සහ Srep වගේ ක්‍රම මඟින් files ඇතුළේ තියෙන සමාන දත්ත (Duplicate data) උපරිම මට්ටමට හඳුනාගෙන ඒවායේ සයිස් එක අඩු කරනවා.

3. මේකේ තියෙන වාසි සහ අවාසි

මේ රීපැක්ස්වල වාසි වගේම අවාසිත් තියෙනවා.

වාසි:

  • Data ඉතිරි වීම: ලංකාවේ අපිට තියෙන ලොකුම ප්‍රශ්නයක් තමයි Internet Data සීමිත වීම. ඉතින් 60GB එකක් 10GB වලින් download කරගන්න ලැබෙන එකම ලොකු වාසියක්.
  • Storage ඉතිරි වීම: හාඩ් ඩිස්ක් එකේ ඉඩ ඉතිරි කරගන්න පුළුවන් වීම.

අවාසි (The Downside):

  • Installation Time වැඩි වීම: මේක තමයි ලොකුම අවාසිය. අර පොඩි කරපු files ආයෙත් නියම ප්‍රමාණයට දිග හරින්න (Decompress කරන්න) ඔයාගේ computer එකට ලොකු වෙහෙසක් දරන්න වෙනවා. සමහර වෙලාවට game එක install වෙන්න පැය ගණනාවක් ගත වෙන්න පුළුවන්. අමුතුවෙන් කියන්න ඕනා නෑනේ experience ඇතිනේ.
  • High CPU සහ RAM භාවිතය: Insall වෙන වෙලාවට ඔයාගේ Computer එකේ Processor එක සහ RAM එක 100%ක්ම වගේ පාවිච්චි වෙනවා. ඒ නිසා පරණ computers වල මේවා install කරන එක ටිකක් අමාරු වැඩක්.
