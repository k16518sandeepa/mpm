Nvidia RTX Spark: PC වලට එන අලුත්ම AI සුපිරි Chip එක ගැන සිංහලෙන්.

Nvidia සමාගම Computex 2026 event එකේදී සාමාන්‍ය පරිගණක (PCs සහ Laptops) සඳහාම විශේෂණය වුණු එයාලගේ අලුත්ම AI SuperChip එක වෙන "Nvidia RTX Spark" නිල වශයෙන් නිවේදනය කරලා තියෙනවා.

Nvidia හි CEO වන Jensen Huang කියන විදිහට, මේක නිකන්ම තවත් ප්‍රොසෙසර් එකක් නෙවෙයි, "පරිගණක කලාවේ 40 වසරක ඉතිහාසය උඩුයටිකුරු කරන නැවත සොයාගැනීමක් (Reinvention)."

Nvidia RTX Spark: Specifications

Nvidia සමාගම මේ සඳහා එයාලගේ දැවැන්ත Data Centers වලට පාවිච්චි කරපු තාක්ෂණය සහ ARM Architecture එක සාමාන්‍ය පරිගණක මට්ටමට ගෙනල්ලා තියෙනවා. මේ චිප් එක නිර්මාණය කරලා තියෙන්නේ තායිවානයේ MediaTek සමාගම සමඟ එකතු වෙලා, TSMC හි අතිශය දියුණු 3-nanometer (3nm) තාක්ෂණයෙන්.

  • GPU : මේකේ අන්තර්ගත වෙන්නේ Nvidia හි අලුත්ම Blackwell ආකෘතිය මත based වුණු GPU එකක්. එකේ 6,144 CUDA Cores සහ පස්වන පරම්පරාවේ Tensor Cores අඩංගු වෙනවා.
  • CPU : NVLink තාක්ෂණය හරහා GPU එකට සෘජුවම සම්බන්ධ කරපු ප්‍රබල 20-core Nvidia Grace CPU එකක් මේකට තියනවා.
  • Unified Memory (RAM): Apple සමාගමේ M-series චිප්ස් වල වගේම මෙහි CPU සහ GPU දෙකටම එකට පාවිච්චි කරන්න පුළුවන් 128GB දක්වා වූ LPDDR5X Unified Memory එකක් තියෙනවා.
  • AI Performance: මේ තනි චිප් එකකට 1 Petaflop (1000 TFLOPS) දක්වා වූ AI පරිගණක බලයක් (FP4 Precision වලින්) ලබා දෙන්න පුළුවන්. මේක දැනට වෙළඳපොලේ තියෙන සාමාන්‍ය AI PC වලට වඩා සිය ගුණයකින් ප්‍රබලයි.

මොකක්ද මේකෙන් කරන්න පුළුවන්? (Capabilities)

මීට කලින් Microsoft Copilot වගේ AI පාවිච්චි කරද්දී ඒ හැම දෙයක්ම run වුණේ Cloud (Internet) එක හරහායි. හැබැයි RTX Spark එක නිසා ඔයාගේ computer එක Local AI Agents run කරන්න පුළුවන්. ඒ කියන්නේ internet නැතුවම pc එක ඇතුළෙන්ම AI එක වැඩ කරනවා.

  1. Agentic AI: ඔයා apps open කර කර, type කර කර ඉන්න ඕන නැහැ. ඔයා pc එකට කියන දේ (Task එක) start එකේ ඉදන් end වෙනකන් තනියම කරන්න පුළුවන් Autonomous AI Agents ලා මේ චිප් එක හරහා ක්‍රියාත්මක වෙනවා. Microsoft සහ Nvidia එකතු වෙලා මේ Agents ලාව Windows Taskbar එකටම pin කරන්නයි යන්නේ.
  2. Frontier LLMs ක්‍රියාත්මක කිරීම: පැරාමීටර් බිලියන 120ක් (120B-parameter) දක්වා ප්‍රමාණයෙන් විශාල Large Language Models (LLMs), ටෝකන් මිලියනයක Context එකක් එක්ක ඔයාගේ ලැප්ටොප් එක ඇතුළෙන්ම රන් කරන්න මේකට පුළුවන්.
  3. Heavy Workloads & Gaming: Creatorsලට 12K වීඩියෝ edit කරන්න, 90GB+ දැවැන්ත 3D visuals Render කරන්න වගේම Gamersලට AAA ගේම්ස් 1440p Resolution එකෙන් 100+ FPS වලින් කිසිම බාධාවක් නැතුව play කරන්න පුළුවන්. Adobe සමාගම දැනටමත් Photoshop සහ Premiere software මේ චිප් එකට ගැළපෙන විදිහට 2x වේගවත් වෙන්න අලුතින්ම නිර්මාණය කරමින් පවතිනවා.

කාටද මේකෙන් තර්ජනයක් වෙන්නේ? (The Market Rivalry)

Nvidia හි මේ පියවරත් එක්ක ඔවුන් ඍජුවම Intel, AMD සහ Apple කියන දැවැන්තයින්ට ලොකු අභියෝගයක් එල්ල කරනවා.

  • Apple සමාගම ඔවුන්ගේ "Unified Memory" සහ ARM architecture එක නිසා battery life එක සහ Performance අතින් ලබාගෙන තිබුණු ඒකාධිකාරය බිඳ වැටෙන්න මේක ලොකු හේතුවක් වෙයි. මොකද Nvidia පවසන විදිහට මේ චිප් එක සහිත Windows ලැප්ටොප් වලට "All-day battery life" (දවස පුරා පවතින බැටරි ආයු කාලයක්) ලැබෙනවා.

තිබිය හැකි එකම බාධාව

තාක්ෂණය උපරිම වුණත්, මේ චිප් එකේ තියෙන Specifications (විශේෂයෙන් 128GB Unified Memory එක) සහ ලෝකයේ පවතින RAM හිඟය නිසා මේ චිප් එක සහිතව එන පළවෙනි models සෑහෙන්න මිල අධික (Premium) ඒවා වෙන්න පුළුවන්.

කවදාද අපිට බලාගන්න පුළුවන්?

මේ අවුරුද්දේ සැප්තැම්බර්/ඔක්තෝබර් (Fall 2026) වෙද්දී Asus, Dell, HP, Lenovo, MSI සහ Microsoft Surface වගේ brands හරහා RTX Spark ඇතුළත් වුණු අලුත්ම Laptops සහ Mini PCs වෙළඳපොළට නිකුත් වීමට නියමිතයි.
