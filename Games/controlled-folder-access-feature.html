අපේ community එකේ ඉන්න ඔයාල ගොඩ දෙනෙක් PC Games play කරල ඇති / කරනව ඇති. ඒ වගේම ඔයා play කරන සමහරක් games save වෙන්නෙ user Documents (C:/Users/username/Documents) folder එකේ වෙන්න ඇති. Controlled Folder Access කියන්නේ මොකක්ද?

Geminiට අනූව Windows 10 Fall Creators Update (Version 1709) (officially released on October 17, 2017) හි ඉදල තමා මේ Controlled Folder Access කියන feature එක නිකුත් වුනේ. මේ feature එකේ අරමුන තමා malware වලින් user documents වෙනස් කිරීම, delete කිරීම ransom තබා ගැනීම වැලක්වීම. ඉතින් මේක usersලට හොද feature එකක් වුනත් වුනත් ඒකෙ ටිකක් ලොකු අවාසියකුත් තිබ්බා. ඒක තමා පරන programs වල save folder එක විදියට user Documents folder එක use කරපු programs වල අර මුලදි කිව්වා වගේ ප්‍රශ්න ඇතිවීම.
